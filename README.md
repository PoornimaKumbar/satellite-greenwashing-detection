Satellite Based Greenwashing Detection System is a Machine Learning and Remote Sensing based project that analyzes satellite images to evaluate environmental conditions. The system calculates indicators such as NDVI, vegetation coverage, and urban presence to estimate a greenwashing score, helping determine whether environmental sustainability claims align with the actual environmental state. An interactive Streamlit dashboard visualizes vegetation patterns, environmental indicators, and risk levels, making environmental monitoring simple and easy to understand. 

**Technologies Used**
Python – Main programming language used to develop the system.
OpenCV – Used for satellite image processing and analysis.
NumPy – Helps in numerical computations and handling image data.
Matplotlib – Used to generate graphs and visualize vegetation data.
Scikit-learn – Used to build and train the machine learning model.
Streamlit – Used to create the interactive web dashboard for displaying results.
Remote Sensing Techniques – Used to analyze satellite images and calculate vegetation indicators like NDVI.
